Caltrain Ridership

Last updated: 2024-10-10

If you're on desktop, Caltrain's new ridership web page has detailed and mostly up-to-date ridership data: https://www.caltrain.com/about-caltrain/statistics-reports/ridership/fare-media-based. Unfortunately that page does not work on mobile.

This site contains graphs and a table of Caltrain's average weekday ridership from Jul 2017 to Sep 2024 sourced from Fare Media Sales-Based Ridership Estimates and JPB Citizens Advisory Committee documents. Average weekday ridership is reported with a 1-month lag, so the most recent data is from Sep 2024.

You can find a CSV of Caltrain's average weekday ridership and the script to generate this site on GitHub. You can find more of my work on my website, echevarria.io.
